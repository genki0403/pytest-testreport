# pytest-testreport

### 1、pytest-testreport介绍

pytest-testreport是一个针对pytest的生成html报告的插件，使用起来非常简单，只需要再pytest.ini文件中做简单的配置即可实现html报告的生成

- 使用案例：安装该插件，在pytest.ini文件加入如下配置，即可实现生成html报告
```ini
[report]
title = 测试报告的标题
tester = 测试人员
desc = 测试报告的说明信息
```
